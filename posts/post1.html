Construyendo mi Portafolio de Trabajo

Publicado el 11 de Julio, 2025

"Si estás en el mundo del desarrollo, sabes que una de las primeras cosas que te dicen es: 'necesitas un portafolio'. Pero, ¿cómo se pasa de una carpeta vacía a un sitio funcional que te represente? Esta no es una guía sobre cómo hacer el portafolio perfecto. Es la crónica real de cómo construí el sitio que estás navegando ahora mismo."

El primer paso

Definir que proyectos quieres presentar es sencillo cuando solo tienes dos de donde escoger, pero a medida que tu experiencia crece, también lo hace la complejidad de tu portafolio. Es importante seleccionar aquellos trabajos que no solo muestren tus habilidades técnicas, sino que también cuenten una historia sobre tu crecimiento como desarrollador. (Espero pronto poder tener ese tipo de dilemas)

El segundo paso

Aqui uno que es nuevo empieza a tener problemas, ¿cómo se hace un sitio web? ¿HTML? ¿CSS? ¿JavaScript? ¿Frameworks? La respuesta es: depende. Para este portafolio, decidí usar HTML y CSS puro, con un poco de JavaScript para la interactividad. No es necesario complicarse con frameworks si lo que buscas es algo simple y efectivo.

El tercer paso

Una vez que tienes el contenido y la estructura, es hora de darle estilo. Aquí es donde entra en juego el diseño. No necesitas ser un diseñador gráfico para hacer algo atractivo. Con un poco de investigación y práctica, puedes crear un sitio que no solo sea funcional, sino también visualmente agradable. Utilicé una paleta de colores simple y tipografías legibles para asegurarme de que el contenido fuera fácil de leer.

El cuarto paso

Finalmente, la parte más importante: la implementación. Una vez que tienes todo listo, es hora de subir tu sitio a un servidor. Puedes usar servicios gratuitos como GitHub Pages o Netlify para empezar. Lo importante es que tu portafolio esté disponible para que otros lo vean y puedan apreciar tu trabajo.

Conclusión

Construir un portafolio no es una tarea fácil, pero con los pasos adecuados y un enfoque en la calidad, puedes crear una herramienta poderosa para mostrar tus habilidades y atraer oportunidades. Recuerda que tu portafolio es un reflejo de ti mismo, así que tómate el tiempo para hacerlo bien.
